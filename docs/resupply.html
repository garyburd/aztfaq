Sierra Vista

Mile: 1.8 nobo / 792.8 sobo Map

Sierra Vista is a gateway community.

Patagonia

Mile: 52.4 nobo / 742.2 sobo Map

Patagonia is a gateway community with grocery stores, lodging, restaurants and a post office.

The town is 4.5 miles south of the trail intersection with AZ 82. Walk in to town on the the Patagonia Train Track trail or contact lodging for shuttle services.

Tucson

Mile: 106.7 nobo / 687.9 sobo (Sahuarita TH) Map
Mile: 112.7 nobo / 681.9 sobo (Gabe Zimmerman TH) Map
Mile: 161.4 nobo / 633.2 sobo (Molino Basin) Map

Tucson is gateway community with many services.

The center of Tucson is approximately 25 miles from the access points listed here. Hikers have had success taking rideshare services from Sahuarita TH and Gabe Zimmerman TH. Sahuarita TH and Molino Basin are the better locations for hitchhiking.

Vail

Mile: 117.4 nobo / 677.2 sobo Map

Vail is a gateway community with grocery stores and a post office.

The Colossal Cave gift shop (near 119.3 nobo / 675.3 sobo) accepts hiker boxes. See the community page for information on sending a package to the gift shop.
